# Métodos de Optimización

OptiLearn Web soporta cinco métodos principales de optimización. El sistema selecciona automáticamente el método más adecuado según las características del problema.

## Reglas de selección automática

El sistema aplica las siguientes reglas en orden:

![Diagrama de Flujo de Selección de Métodos](/images/diagramas/diagrama_flujo.png)

1. **¿Menciona proceso iterativo?** → Gradiente Descendente
2. **¿Hay restricciones no lineales?** → KKT
3. **¿Solo restricciones de igualdad?** → Lagrange
4. **¿Función cuadrática + restricciones lineales?** → Programación Cuadrática (QP)
5. **¿Pide derivadas?** → Cálculo Diferencial
