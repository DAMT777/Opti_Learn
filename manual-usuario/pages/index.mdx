# Manual de Usuario - OptiLearn Web

Bienvenido al manual de usuario del sistema **OptiLearn Web**, una plataforma educativa para la resolución de problemas de Programación No Lineal con asistencia de inteligencia artificial.

## ¿Qué es OptiLearn Web?

OptiLearn Web es un sistema que permite resolver problemas de optimización matemática de forma interactiva, proporcionando explicaciones pedagógicas paso a paso y visualizaciones del proceso de solución.

## Características principales

- **Análisis automático de problemas**: El sistema detecta automáticamente el método de optimización más adecuado.
- **Múltiples métodos de solución**: Soporta Gradiente Descendente, Lagrange, KKT, Programación Cuadrática y Cálculo Diferencial.
- **Asistente IA**: Integración con LLaMA para explicaciones didácticas y recomendaciones.
- **Interfaz web intuitiva**: Fácil de usar, no requiere conocimientos de programación.

## Navegación del manual

Utiliza el menú lateral para explorar las diferentes secciones:

- **Introducción**: Información general del sistema
- **Primeros pasos**: Cómo comenzar a usar OptiLearn Web
- **Métodos de optimización**: Guías detalladas de cada método
- **Ejemplos**: Casos prácticos resueltos paso a paso
- **Arquitectura**: Detalles técnicos del sistema
- **FAQ**: Preguntas frecuentes
