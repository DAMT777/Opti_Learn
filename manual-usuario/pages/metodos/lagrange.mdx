# Multiplicadores de Lagrange

## ¿Cuándo se usa?

El método de Multiplicadores de Lagrange se usa cuando:
- El problema tiene **solo restricciones de igualdad**
- No hay restricciones de desigualdad
- Se busca optimizar una función diferenciable

## Fundamento matemático

El método construye la función Lagrangiana:

$$
L(x, \lambda) = f(x) + \sum_{i=1}^m \lambda_i g_i(x)
$$

Donde:
- $f(x)$ es la función objetivo
- $g_i(x) = 0$ son las restricciones de igualdad
- $\lambda_i$ son los multiplicadores de Lagrange

## Condiciones de optimalidad

Para encontrar el óptimo, se resuelve el sistema:

$$
\begin{aligned}
\nabla_x L &= 0 \\
g_i(x) &= 0, \quad \forall i
\end{aligned}
$$

## Ejemplo

![Ejemplo de Multiplicadores de Lagrange](/images/screenshots/ejemplo-lagrange.png)

**Problema**: Minimizar $f(x, y) = x^2 + y^2$ sujeto a $x + y = 5$

**Entrada**:
```
Minimizar f(x,y) = x^2 + y^2
sujeto a:
  x + y = 5
```

**Proceso**:
1. Formar el Lagrangiano: $L(x, y, \lambda) = x^2 + y^2 + \lambda(x + y - 5)$
2. Calcular derivadas parciales:
   - $\frac{\partial L}{\partial x} = 2x + \lambda = 0$
   - $\frac{\partial L}{\partial y} = 2y + \lambda = 0$
   - $\frac{\partial L}{\partial \lambda} = x + y - 5 = 0$
3. Resolver el sistema de ecuaciones
4. Verificar con el Hessiano restringido

**Resultado**:
- Punto óptimo: $x^* = [2.5, 2.5]$
- Valor mínimo: $f^* = 12.5$
- Multiplicador: $\lambda^* = -5$

## Interpretación de los multiplicadores

El valor de $\lambda_i$ indica:
- La sensibilidad del óptimo respecto a cambios en la restricción $i$
- Si $|\lambda_i|$ es grande, la restricción es "activa" o importante
- El signo indica si relajar la restricción mejora o empeora el óptimo

## Parámetros importantes

- **equalities**: Lista de restricciones de igualdad (automático)
- **variables**: Variables del problema (automático)
